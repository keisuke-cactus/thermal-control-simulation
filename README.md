## 背景
長尺半田槽では複数ヒーターを配置して温度均一化を行う。しかし、各制御ブロック間には熱伝導による相互作用が存在し、独立したPID制御のみでは温度分布が不安定になる懸念があった。

## 目的

隣同士の温度干渉をモデル化した制御プログラムを構築し、長尺半田槽温度を均一化する。

## 要求仕様

半田温度　320℃±20℃
半田交換後30分以内に安定化
外乱（製品親戚による温度低下）発生後＿分以内に温度が収束
